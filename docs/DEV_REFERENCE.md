# Git Quick Reference

## Check repository status

```bash
git status
```

## Add files to staging

```bash
git add filename
```

## Commit staged files

```bash
git commit -m "message"
```

## Push changes to GitHub

```bash
git push
```

## View commit history

```bash
git log --oneline
```

## Visual history

```bash
git log --graph --decorate --oneline
```

## Typical Git workflow

```text
edit file
git add
git commit
git push
```
